# Seentrix CRA Translation Glossary (EN · DE · FR · IT)

Canonical translations for CRA/regulatory and product terms. Every translator
(human or AI) MUST use these exact renderings so terminology stays consistent
and legally correct across the UI, Academy, Copilot, and documents. When a term
below appears in a source string, translate it per this table.

## Rules
- Keep **product/brand names** untranslated: "Seentrix", "SBOM", "CycloneDX",
  "SPDX", "CE", "CVSS", "EPSS", "CVE", "PSIRT", "ENISA", "CSIRT", "API".
- Keep the regulation's short name "CRA" as-is; spell-out follows the table.
- Preserve placeholders exactly: `{count}`, `{name}`, `<b>…</b>`, `{0}`, etc.
- Match source capitalization style and punctuation; don't add trailing periods
  that aren't in the source.
- Use formal address: German **Sie**, French **vous**, Italian **Lei/voi** as
  appropriate for professional B2B SaaS.
